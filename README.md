# 🏭 Steel Surface Defect Detection

## 📌 Project Overview
This project uses Deep Learning and Computer Vision techniques to automatically detect and classify defects on steel surfaces from industrial images. Automated defect detection helps improve quality control, reduce production costs, and increase manufacturing efficiency.

## 🎯 Objectives
- Detect defects on steel surfaces.
- Classify different defect types accurately.
- Improve industrial quality inspection using AI.

## 📂 Dataset
The dataset contains steel surface images with different defect categories such as:
- Crazing
- Inclusion
- Patches
- Pitted Surface
- Rolled-in Scale
- Scratches

## 🛠 Technologies Used
- Python
- Jupyter Notebook
- TensorFlow/Keras
- NumPy
- Pandas

- Matplotlib
- Scikit-learn

## 🚀 How to Run
1. Clone the repository.
2. Install the required libraries.
3. Open the Jupyter Notebook.
4. Run all cells to train or test the model.

## 📊 Model
- Convolutional Neural Network (CNN)
- Optimizer: Adam
- Loss Function: Categorical Crossentropy
- Evaluation Metric: Accuracy

## 📈 Results
The trained model successfully detects and classifies multiple types of steel surface defects with high accuracy.
